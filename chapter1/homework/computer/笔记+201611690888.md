
# python
## 第一节课堂笔记
### Python的安装
- 打开https://www.python.org/ftp/python/3.6.0/python-3.6.0-amd64-webinstall.exe
- 安装到D:\python下（勾选 Install for all users以及add python to Path）
- Win+R 回车
- 键入powershell 回车
- 键入python

### jupyter notebook 安装
- Win+R 回车
